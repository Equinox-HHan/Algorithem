# 暑假算法学习笔记

## 阶段1：

### day1-2:
1. 主定理：用于求解分治算法递推关系式时间复杂度：T(n)=aT(n/b)+f(n)
2. 三码问题：
    - 正数的原码反码补码都是一致的
    - 负数反码是原码除符号位全取反，补码是原码取反加一
    - 负数补码取反加一或者减一取反得到原码
    - 所有编码都是0正1负
3. 位运算：
    - A^0=A,A^A=0（xor相同取0不同取1,满足交换律和结合律）
    - n&(n-1)：去除二进制最右边的1
    - n&(-n)：获取二进制最右边的1
    - ~n&(n+1)：获取最右边的0
    - 进行乘除运算和奇偶判断

### day3-4:
1. GCD(GCD(a,b)==GCD(b,a%b)),LCM(GCD(a,b)×LCM(a,b)=|a×b|)
2. 快速幂：时间复杂度logn
3. 埃氏筛：素数的倍数是合数（两个优化）
4. 同余原理（除法是特例）

### day5-6:
1. 基础二分查找
2. 寻找首末位特殊二分查找
3. 山峰位置二分查找`l<r`


## 阶段2：

    2. 链表找中点或者找环
    3. 滑动窗口：连续子数组，连续子串
