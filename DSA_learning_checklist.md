# 🎯 数据结构与算法 (DSA) 核心学习与打卡清单

> **说明**：本清单专为考研与大厂面试定制，剔除了 ACM 竞赛级的偏难怪题（如线段树高级应用、网络流、高级数论等），聚焦于传统且高频的计算机科学基本盘。建议结合“艾宾浩斯记忆法”循环复习。
> **重要度图例**：
> ⭐⭐⭐：面试/笔试极高频，必须闭眼手撕（完全形成肌肉记忆）。
> ⭐⭐：常见考点，需熟练掌握基本模板和变体。
> ⭐：低频或理论为主，重在理解思想，能写出基础版即可。

***本学期教材目录： https://www.tup.com.cn/booksCenter/book_08253101.html ***

## 模块一：复杂度与基础语言技巧
- [ ] ⭐⭐⭐ **时间与空间复杂度分析**：掌握大 O 表示法，了解主定理 (Master Theorem)，能一眼看透代码的复杂度。
- [ ] ⭐⭐ **位运算技巧**：异或的性质（找奇数次的数、不使用额外变量交换两数）、提取最右侧的 1（`x & -x`）、消除最右侧的 1（`x & (x-1)`）、基础位图 (BitMap)。
- [ ] ⭐⭐ **基础数学与数论**：最大公约数 (GCD) 与最小公倍数 (LCM)、快速幂算法、素数筛法（埃氏筛）。
- [ ] ⭐⭐⭐ **对数器（左程云核心体系）**：学会自己写随机数据生成器和暴力解，用于验证最优解算法的正确性（摆脱对在线评测系统的绝对依赖）。

## 模块二：基础数据结构（线性表）
### 1. 数组、哈希表与矩阵
- [ ] ⭐⭐⭐ **二分查找**：基础二分、寻找左右边界的二分（模板必须烂熟于心）、局部最小值问题、二维矩阵的二分查找。
- [ ] ⭐⭐⭐ **双指针**：相向双指针（回文、两数之和、接雨水）、同向双指针（快慢指针、移动零）。
- [ ] ⭐⭐⭐ **滑动窗口**：定长窗口、不定长窗口（寻找最长/最短满足条件的子串）、包含/排列异位词问题。
- [ ] ⭐⭐⭐ **前缀和与差分**：一维/二维前缀和、一维差分数组（区间加减操作）。
- [ ] ⭐⭐⭐ **哈希表与高频技巧**：两数之和系列、摩尔投票法 (Majority Element，找众数)、原地哈希 (寻找消失的数字/缺失的第一个正数)。
- [ ] ⭐⭐ **矩阵模拟**：螺旋矩阵、旋转图像、对角线遍历。
### 2. 链表
- [ ] ⭐⭐⭐ **基础操作**：单链表/双链表的节点定义、插入、删除、虚拟头节点 (Dummy Node) 技巧。
- [ ] ⭐⭐⭐ **链表反转**：反转整个链表、反转部分链表 (区间反转)、K 个一组反转链表（高频手撕）。
- [ ] ⭐⭐⭐ **快慢指针**：寻找中点、判断环、寻找入环节点、两个无环/有环单链表的相交节点。
- [ ] ⭐⭐ **链表合并与拆分**：合并两个有序链表、合并 K 个有序链表（结合堆）、奇偶链表分离。
- [ ] ⭐⭐⭐ **经典数据结构设计**：**LRU 缓存机制 (极其高频，必考双向链表+哈希表)**、LFU 缓存机制（辅助了解）。
### 3. 栈与队列
- [ ] ⭐⭐⭐ **基础实现**：数组实现栈和队列、栈和队列的相互实现、最小栈 (Min Stack) 的设计。
- [ ] ⭐⭐⭐ **单调栈**：寻找数组中每一个数左/右侧最近的比它大/小的数、接雨水问题、柱状图中最大的矩形（极其重要，左程云高频强调）。
- [ ] ⭐⭐ **单调队列**：滑动窗口最大值问题。
- [ ] ⭐⭐ **表达式求值 (408常考)**：中缀表达式转后缀表达式、后缀/逆波兰表达式求值。

## 模块三：排序算法专题
> 面试常常直接要求手写，或者借用其核心思想（如归并思想、快排 Partition 思想）。
- [ ] ⭐ **基础排序**：冒泡排序、选择排序、插入排序（重在理解其稳定性与时间复杂度边界）。
- [ ] ⭐⭐⭐ **归并排序**：递归实现、非递归实现、利用归并排序求“小和”问题/逆序对问题。
- [ ] ⭐⭐⭐ **快速排序**：荷兰国旗问题（Partition 思想）、随机快排的实现与复杂度分析、**快速选择算法 (Quick Select 解决 Top K 问题)**。
- [ ] ⭐⭐⭐ **堆排序**：手写 `heapInsert` 和 `heapify`，理解完全二叉树的数组映射。
- [ ] ⭐ **非比较排序**：计数排序、基数排序、桶排序（了解思想及适用场景）。

## 模块四：树与高级数据结构
### 1. 二叉树
- [ ] ⭐⭐⭐ **树的遍历**：前、中、后序的递归与**非递归实现**（压栈法）、层序遍历（队列）。
- [ ] ⭐⭐ **Morris 遍历 (左程云特色)**：利用线索二叉树实现 O(1) 空间复杂度的遍历。
- [ ] ⭐⭐⭐ **树的属性求值**：求最大深度/直径、判断平衡二叉树、判断对称二叉树、最近公共祖先 (LCA)、路径总和系列。
- [ ] ⭐⭐⭐ **二叉树的序列化与反序列化**：以前序遍历或层序遍历为准则进行序列化。
- [ ] ⭐⭐⭐ **二叉搜索树 (BST)**：BST 的判断、节点的增删改查、利用 BST 中序遍历有序的性质。
- [ ] ⭐⭐⭐ **二叉树的递归套路（树形 DP 基础）**：左树要什么信息，右树要什么信息，如何合并信息（左程云核心解题套路，如判断满二叉树、最大二叉搜索子树）。
### 2. 堆 (Heap) 与 字典树 (Trie)
- [ ] ⭐⭐⭐ **优先级队列 (PriorityQueue)**：熟练使用系统内置堆解决 Top-K 问题、数据流中位数问题、合并 K 个有序链表。
- [ ] ⭐⭐ **字典树 (Trie)**：前缀树的数组/哈希表实现，解决多字符串的前缀匹配问题。
### 3. 并查集 (Disjoint Set Union)
- [ ] ⭐⭐⭐ **并查集模板**：数组实现、路径压缩 (`find` 优化)、按秩合并/节点数量合并 (`union` 优化)。
- [ ] ⭐⭐ **应用场景**：连通块数量统计、判断图中是否有环、岛屿数量变种。
### 4. 进阶树结构（大厂笔试/408加分项）
- [ ] ⭐ **线段树 (Segment Tree)**：区间更新与区间查询（仅需掌握基础的单点/区间加法模板）。
- [ ] ⭐ **树状数组 (Binary Indexed Tree / Fenwick Tree)**：单点修改与区间查询。

## 模块五：图论 (Graph)
> 面试和 408 常考图的经典算法，不需要钻研过于复杂的图论。
- [ ] ⭐⭐ **图的表示法**：邻接矩阵、邻接表、链式前向星（以及如何将不同题目给的图结构转换为自己熟悉的模板结构）。
- [ ] ⭐⭐⭐ **图的遍历**：深度优先遍历 (DFS)、广度优先遍历 (BFS)、二分图判定（染色法）。
- [ ] ⭐⭐⭐ **拓扑排序**：判断有向图是否有环、课程表问题（入度表法 BFS，以及 DFS 实现）。
- [ ] ⭐⭐ **最小生成树 (MST)**：Kruskal 算法（结合并查集）、Prim 算法。
- [ ] ⭐⭐ **最短路径**：Dijkstra 算法（普通版与堆优化版）、Bellman-Ford / SPFA 算法（处理负权边）、Floyd 算法（多源最短路，408常考）。

## 模块六：暴力尝试与动态规划 (DP)
> 左程云强调：一切 DP 皆来自于暴力的记忆化搜索。必须从 DFS 起步。
### 1. 递归与回溯 (DFS)
- [ ] ⭐⭐⭐ **子集与排列组合**：全排列（无重复/有重复）、组合总和、子集。
- [ ] ⭐⭐⭐ **二维网格 DFS**：岛屿数量问题、单词搜索、N 皇后问题、解数独。
### 2. 动态规划基础 (从尝试到递推)
- [ ] ⭐⭐⭐ **记忆化搜索**：深刻理解从“纯暴力 DFS” 加上“缓存 (Memoization)” 进化到 DP 的过程。
- [ ] ⭐⭐⭐ **一维线性 DP**：斐波那契数列变种、爬楼梯、打家劫舍系列、最大子数组和。
- [ ] ⭐⭐⭐ **二维网格 DP**：不同路径、最小路径和、下降路径最小和。
- [ ] ⭐⭐⭐ **序列 DP**：最长递增子序列 (LIS，含 O(NlogN) 二分查找解法)、最长公共子序列 (LCS)、编辑距离。
### 3. 背包问题专题
- [ ] ⭐⭐⭐ **0-1 背包**：核心模板及空间压缩（一维数组）、分割等和子集问题。
- [ ] ⭐⭐⭐ **完全背包**：零钱兑换系列。
- [ ] ⭐ **多重背包**：了解二进制拆分优化思想。
### 4. 进阶 DP（笔试极高频，面试看情况）
- [ ] ⭐⭐ **状态机 DP**：股票买卖系列（掌握通解框架）。
- [ ] ⭐⭐ **区间 DP**：最长回文子串/子序列、戳气球、矩阵连乘（408常考）。
- [ ] ⭐ **树形 DP**：打家劫舍 III（没有上司的舞会）。
- [ ] ⭐ **数位 DP**：统计 1 的个数等（作为笔试防线，掌握一种通用模板即可）。

## 模块七：贪心算法与其他
- [ ] ⭐⭐ **贪心策略**：区间调度问题（最多参加多少个会议、合并区间）、跳跃游戏、哈夫曼编码树思想。
  *注：贪心算法往往没有通用模板，重点在于多刷题培养直觉，或者通过对数器验证贪心策略。*
- [ ] ⭐⭐ **字符串基础**：字符串反转、替换空格、整数与字符串相互转换。
- [ ] ⭐ **高级字符串算法**：
  - **KMP 算法**：核心思想及 `next` 数组求解（考研重点，面试常考思想）。
  - **字符串哈希 (Rabin-Karp)**：快速比较字符串子串。
  - **Manacher 算法 (马拉车)**：O(N) 解决最长回文子串（左程云特色，大厂笔试偶考）。
